canDeactivate(transition) [-> Promise | Boolean]

离开的组件在验证时期，会调用此钩子函数

  • 参数 Arguments transition {Transition}

    调用 transition.next()来解决此钩子函数，这里调用 transition.abort() 的时候，就可以取消过渡了。

  • 返回值 Return Value 可以选择返回一个 Promise
    • resolve(true) -> transition.next()
    • resolve(false) -> transition.abort()
    • reject(reason) -> transition.abort(reason)
    或者 选择 返回一个 Boolean 值
    • true -> transition.next()
    • false -> transition.abort()
  • 详细信息 Detail

    此钩子函数调用从下到上，一个父视图组件的 canDeactivate 只会在 子视图组件的 canDeactivate调用完毕后调用
