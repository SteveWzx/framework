canReuse: Boolean | canReuse(transition) -> Boolean

确定组件是否可以重新使用

如果一个组件不能被重用，那么当前实例将被一个新的取代，它将进入正常的验证和激活阶段

此canReuse 可以是一个Boolean 值或者是一个函数，同步的返回 一个 Boolean值，默认为 true

  • 参数 Arguments transition {Transition} 和 active一样

    你只能访问 transition.to 和 transition.from 在此钩子函数中。

  • 返回值 Return Value 必须返回一个 Boolean值， 假值会被视为false
  • 详细信息 Detail

    canReuse 会被同步调用， 在所有潜在可以重用的组件按照自上而下的顺序。

    如果一个组件可以被重用，它的 data 钩子函数 会在激活阶段被调用
