回首页Route Options api 地址

hashbang：(此demo页面已设置l hashbang:false)

  • var router = new VueRouter({hashbang:false})
  • hashbang：设置为false的时候，注意地址栏变化,没有感叹号了

history：启用html5 的history模式，但是需要 在服务端设置一些规则 properly configured

abstract：开启后 地址栏就不会变了

root：只能用在html history模式中 ，不过大多数情况你不用关心它

saveScrollPosition：只能用在html history模式中 ，切换的时候保持滚动条之前的位置，如果失效，可能是因为<router-view> 有过度效果

linkActiveClass：链接样式 默认：v-link-active ， v-link-active 类名，如果这个地址匹配那么就加上 v-link-active-exact

transitionOnLoad： 初始化的时候无论给<router-view>何种过度效果 ，默认的组建会瞬间呈现

suppressTransitionError： 如果设置为true，那么过渡效果中的错误就不会提示

Route Object & Route Matching

使用router.go('/user?user=1') 跳转

/a /a/b Go to /user?query --> /user?user=123 Go to /user/:username --> /user/lilei Go to /user/:username/post/:post_id --> /user/evan/post/123 Go to /user/*any --> /user/a/b/c Go to /foo/*any/bar --> /foo/a/b/bar
